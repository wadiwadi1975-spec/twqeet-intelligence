# TWQEET Intelligence Platform

![TWQEET Logo](frontend/public/logo.svg)

منصة ذكية لإدارة محلات بيع الذهب والمجوهرات

## الميزات

- إدارة المبيعات والمشتريات
- تحليل المخزون بالعيارات والفئات
- مقارنة أداء الفروع
- إدارة العملاء والموردين
- التقارير المالية (PDF/Excel/CSV)
- الذكاء الصناعي والتوصيات
- التنبيهات الذكية
- لوحة الإدارة العليا

## التشغيل

### عبر Docker
```bash
docker-compose up
```

### يدويًا
```bash
# Backend
cd backend
npm install
npm run start:dev

# Frontend
cd frontend
npm install
npm run dev
```

## الروابط
- Frontend: http://localhost:3000
- Backend: http://localhost:4000
- MongoDB: localhost:27017
